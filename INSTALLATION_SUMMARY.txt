🎯 HOW IT WORKS:
═══════════════════════════════════════════════════════════════════════════════

BEFORE (Complex - 6 steps):
  git clone repo → cd repo → setup.sh → activate venv → 
  set API key → python agent.py --tui

AFTER (Simple - 3 steps):
  1. Download and run install.sh from GitHub
  2. Set API key: export ANTHROPIC_API_KEY="..."
  3. Run anywhere: cd ~/any-project && mimicode

The installer:
  • Clones mimicode to ~/.mimicode
  • Creates global wrapper in /usr/local/bin or ~/.local/bin
  • Auto-creates venv and installs dependencies
  • Lets you run `mimicode` from any directory
  • Agent works on files in your current directory
